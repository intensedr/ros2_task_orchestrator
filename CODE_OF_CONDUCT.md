# Code Of Conduct

This project is committed to a respectful, inclusive and practical community
environment.

Expected behavior:

- Be respectful and constructive.
- Assume good intent, but be clear when something needs to change.
- Keep technical disagreements focused on evidence and tradeoffs.
- Do not harass, threaten or discriminate.

Maintainers may remove comments, issues or pull requests that violate these
expectations.
